GPS Tracker

Internship Projects aimed to build a portable and power efficient GPS tracker

Visualize on Google Maps

Built with Microchip Technology AVR © devices

The project is built upon the cost and energy efficient AVR 8-bit microcontrollers offered by Microchip Technology, ensuring sufficient power in a small package. The project is build with good compatibility and portability in mind, the source code being written in Microchip's MCC Melody code configurator.

Compact and Convenient

The GPS Tracker comes in a small 10x10cm package that can be easily powered by one Li-Ion battery in 18650 format, rechargable through the available micro-USB port.

Simple and effortless

Recieved GPS coordinates will be rendered in a user friendly manner, using the Google Maps API.
